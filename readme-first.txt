this tool will migrate jbpm3.3 diagram to jbpm5 version. this tool will only migrate the graphical model, 
it does not include any code of the jbpm3.3 model.

the tools used are:
- eclipse indigo for jbpm5
- jbpm5 and jbpm5.4.0 for eclipse


the main program is Migrate.java, the source and destination directory is set here as per my needs.
you may change this as you see fits.
